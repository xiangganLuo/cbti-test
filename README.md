# CBTI（Cat‑Based Temperament Indicator）· 人格测试

> 轻量网页 / H5 单页应用。18 题 → 自动生成 16 种结果之一的图片并可一键保存。本测试仅供娱乐，无科学依据。

## 功能概览
- 打开即用：无需登录/授权/手机号。
- 答题流程：18 题，单题单页，点击即下一题，无回退。
- 自动出图：答完第 18 题立即生成结果竖图（1080×1920），支持保存到本地。
- 不含分享：无朋友圈/微博/小红书等跳转，无广告，无外链。

## 代码位置
本仓库实现放在目录：`../cat-mind-test/`

- `index.html`：单页应用（首页/答题/结果）。
- `styles.css`：样式。
- `app.js`：业务逻辑（题库加载、计分、结果解析、Canvas 出图）。
- `assets/questions.json`：题库（18 题，A/B/C/D）。
- `assets/results.json`：结果标签库（16 类型，含配色与「猫」风格说明）。

> 说明：此 `cbti-test` 目录仅用于文档与规范；如需运行请进入 `cat-mind-test`。

## 本地预览
```bash
cd ../cat-mind-test
python3 -m http.server 5173
# 浏览器打开 http://localhost:5173
```

## 使用说明
1) 点击「开始测试」进入答题；2) 18 题全部作答后自动跳到结果页并生成图片；3) 点击「保存图片」即可下载 PNG。

## 自定义与二次开发
- 修改题库：编辑 `assets/questions.json`。
- 修改结果外观：在 `assets/results.json` 中调整 `palette`（配色）与 `catStyle`（装饰/姿态）。
- 导出分辨率：在 `app.js` 的 `CanvasCard` 构造函数中调整 `w/h`（默认 1080×1920）。
